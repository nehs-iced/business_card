社團介紹

學習、研究、分享資訊技術

🛠️ 開發組（Python）｜打造專屬網站與 Discord 機器人

想親手打造一個屬於自己的網站？或是讓你的 Discord Server 擁有專屬機器人？ 開發組將帶你從零開始掌握 Python Flask 網站框架 與 Discord Bot 開發技巧！ 不只學理論，還會手把手帶你實作專案，從網站部署到機器人互動通通包辦。 無論是初學者還是已經有基礎的你，都能在這裡找到成長與挑戰！

  • 社課地點：活動中心底下的小閱覽室
  • 因場地無電腦，第二堂課起請自備筆電

⏱️ 競賽組（C++）｜用演算法征服競賽舞台

從零開始Coding？對解題有熱情？想了解資訊科技（Information Techonology）的奧秘？ 競賽組專注於程式競賽技巧與思維養成，講授演算法與資料結構， 從基礎到進階教你怎樣解題。搭配不定期社內賽，讓你實戰的能力不斷提升！無論是準備 APCS、校內選拔， 還是想進軍資奧、NOI，這裡就是你磨練技術、挑戰極限的最佳舞台！

  • 社課地點：科藝大樓二樓 國中資訊教室
